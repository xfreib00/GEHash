Detailed Description

template<typename T, typename V, typename = std::enable_if_t<std::is_unsigned_v<T>>>
class GEHash::HTable< T, V, typename >

Class implementing basic hash table.

Class is set up as hash table with customizable hash function, allowing us to dynamically change hash function when needed. Hash function is provided in form of string via setFunc function. Implicitly table has no hash function.

Template Parameters
TData type specifying range of hash table indexes.
VData type of records.
